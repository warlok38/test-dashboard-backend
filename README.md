# Test Dashboard Backend

Local NestJS API for the Polyus test dashboard.

## Scripts

```powershell
npm.cmd run start:dev
npm.cmd run build
npm.cmd run lint
npm.cmd test
```

## Local URLs

- API base: http://localhost:3001/api
- Swagger: http://localhost:3001/api/docs
- Health: http://localhost:3001/api/health
